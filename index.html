Example 1

This is a visualization that reflects the distibution of dietary preferences across India. The visualisation uses concentric charts within the state lines to represent data. While the map with a pie chart is a regularly used way to visualise data, using concentric circles map might not be the best way to represent data that are typically not considered to be a subsect of the larger dataset.


Source: Huffpost India


Analysing some issues within this visualisation:


Idiom:

  1. The visualisation use a concentric circle map, while this representation suggests the idea that vegetarians are part of non-vegetarians. While this is technichally a true statement, it is not a common interpretation on the vegetarian population who are typically considered a seperate categorization. The data represented is the percentage of Vegetarian and non-vegetarian population in each state, which does not translate well to this data type, hence we see most states need the percentages indicated via arrows outside the state.

  2. The colours chosen also make an unintentional statement, as green and red suggests one category is more positive, and the other is negative. The visualisation needs to state in legend that it represents percentages, rather than it being obvious when analysing the visuals.

Improvement:

  1. The data could be represented as a chloropleth with interactive elements that represent the percentage distribution, or by using pie chart instead on the concentric circle map.

  2. The colours chosen might be more neutral colours, that do not associate values to the data.

  3. Represent the state names by their state's name, rather than the state abbreviation, so people not familiar with Indian state names are able to interpret the data.

  4. Title does not convey enough information, could use a sub-heading to make it clearer before reading the legend.

Example 2

This is a visualisation that represents the percentage of world debt of different countries. The infographic, looks at two major measures: (1) Share of global debt as a percentage, and (2) Debt-to-GDP. The concept of what the debt is compared against is unclear. The placement of the countries within this representation itself is very confusing, and not intuitive, as it is impossible to find countries easily, or understand the intention of the placement.


Source: World Economic Forum


Analysing some issues within this visualisation:


Idiom:

  1. The visualization uses hexagonal binning plot within a circular boundary to represent the data, but it is unclear what the area of the hexagons represent and is not explained anywhere properly.

  2. The visualization also uses a chloropleth to indicate percentage values of debt, but given that the countries are not positioned in any specific way, it is hard to compare different countries by any metrics.

  3. The colours used are more useful for heatmaps, a graduated/gradient colour scheme might give a better idea at first glance.

Improvement:

  1. If this kind of data representation is used, the data could organised from most to least dept as percentage of GDP.

  2. This data could be represented on a world map so it is easier to identify and compare countries debts. This would also add an additional element of information to the data visualisation.

  3. Could benefit from interactivity, to offer more context for the data, and compare data points with ease.
